# Tradify

Cette application permet d'identifier une pièce de musique traditionnelle québécoise ou irlandaise du répertoire montréalais.

L'application est disponible [ici](https://ludovical.github.io/Tradify/).


Voici ce qu'il reste à faire:

- Transformer les recherches en objets pour les rendre plus faciles à manipuler et à passer entre les fonctions
- Construire une liste de pièces que l'application ne doit pas inclure dans l'index
- Permettre à l'utilisateur de soumettre un fichier wav pour analyse
